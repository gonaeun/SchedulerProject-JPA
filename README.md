## 📌 프로젝트 개요
- **목표** : Spring Boot 기반 애플리케이션의 코드 품질 향상과 테스트 커버리지 확보를 목적으로 진행되었습니다. 실무에서 발생할 수 있는 다양한 문제를 사전에 인식하고, 구조적 개선을 통해 유지보수성과 성능을 높이는 것을 목표로 합니다.

- **기술스택** : Java, Spring Boot
